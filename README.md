# Vue school showdown
This is an exercise that consists of a set of tasks. Itâ€™s designed to showcase your proficiency in leveraging your Tailwind, Vue, and Nuxt skills to transform a Figma design into a dynamic website.

## Setup

Make sure to install the dependencies:

```bash
# pnpm
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# pnpm
pnpm run dev
```

## Production

Build the application for production:

```bash
# pnpm
pnpm run build

```

Locally preview production build:

```bash
# pnpm
pnpm run preview

```
